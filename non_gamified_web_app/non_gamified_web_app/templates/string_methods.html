    • The upper() method converts a string into uppercase

  • Replace
    • The replace(a,b) function takes in two arguments
    • The first argument is the word/phrase that needs to be replaced
    • The second argument is the word/phrase that will replace the first argument
  • opinion1 = “Python is hard”
    opinion2 = opinion1.replace(“hard”, “fun!”)
    opinion2 = “Python is fun!”
  • Find
    • The find(a) method finds the first occurrence of a specific string
    • It returns an integer value of the position
    • This method returns -1 if the string value is not found
  • opinion = “Python is fun”

    opinion.find(“fun”) = 10
